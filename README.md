# Bloomwater

Landing page for Bloomwater — handcrafted bath bombs made in small batches.

**Live site:** https://zuvv.github.io/bloomwater-site/

## Structure

```
index.html        Main page
css/styles.css    All styles
js/main.js        Animation logic
```

## Stack

Plain HTML, CSS, and JavaScript. No build step. Hosted on GitHub Pages from the `main` branch.
